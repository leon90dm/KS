# 时间序列

## 基本定义

??? info classes "定义"
     1. $\rho_{l}=\frac{\operatorname{Cov}\left(r_{t}, r_{t-l}\right)}{\operatorname{Var}\left(r_{t}\right)}$  间隔 $l$ 的自相关系数。

## 平稳性

=== "严平稳"
    ($r_1,\dots,r_t$) 的联合分布在时间的平移变换下 **保持不变**。

=== "弱平稳"
    1. 一阶矩为常数，$E(r_t) = u$ 均值为常数。
    2. 二阶矩收敛，$E(r_t^2) < \infty$.
    3. $Cov(r_t, r_{t-l}) = \gamma_l$ 结果只依赖于 $l$ ，而不依赖于 $t$
