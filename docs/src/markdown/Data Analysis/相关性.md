## 自相关

给定一个时间序列数据 $\{r_t\}_{t=1}^T$ , $\bar r$ 为均值， 则间隔为1（$l=1$）的自相关系数为

$$
\hat{\rho}_{1}=\frac{\sum_{t=2}^{T}\left(r_{t}-\bar{r}\right)\left(r_{t-1}-\bar{r}\right)}{\sum_{t=1}^{T}\left(r_{t}-\bar{r}\right)^{2}}
$$
